# Lulama Nsizwa Zondo - Professional Portfolio

A sleek, responsive, modern developer portfolio showcasing my skills as an IT Graduate and Web Developer. This website highlights my background, personal projects, professional credentials, and provides an effortless way to get in touch.

## 🚀 Features

*   **Dynamic Landing Page:** Features a typed heading animation and clean overlay aesthetics.
*   **Fully Responsive Navigation:** Optimized for mobile viewports using a toggleable hamburger slide-out drawer menu.
*   **Smooth Scrolling Sliders:** Implements custom CSS layout configurations designed for horizonal carousel scrolling sections (Projects & Certificates).
*   **Integrated Credentials:** Dynamic tracking linking directly to verified Microsoft Azure and Coursera milestones.
*   **Accessible Call to Actions:** Direct links for CV downloads, one-click email triggers, and WhatsApp redirection.

---

## 🛠️ Built With

*   **HTML5:** Semantic markup structure ensuring accessibility across devices.
*   **CSS3:** Flexbox layouts, media query break-points, smooth layout transitions, and bespoke keyframe typography animations.
*   **JavaScript (ES6):** Controls DOM manipulations for mobile menus and individual carousel slider scrolling configurations.
*   **FontAwesome:** Lightweight iconography scaling cleanly across resolution profiles.
